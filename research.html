01 // SELECTED PROJECT
EEG STUDY NEURAL SYNCHRONY AUDACITY TRF ANALYSIS

How Consonant and Dissonant Music affect Neural Synchrony

Using ten house music tracks with almost the same BPM (123), modulated into better and worse superposition versions using Audacity. Collected subjects' tapping synchrony to the beats and EEG data, then used Temporal Response Function (TRF) to test the music-brain synchrony.
